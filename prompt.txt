Actúa como un agente senior de desarrollo frontend especializado en HTML, CSS, accesibilidad, diseño responsive y organización de proyectos web estáticos.

OBJETIVO

Crear la Interfaz de Acceso o Portada principal de un proyecto web para una práctica de Prompt Engineering en Agentes de IA.

El proyecto debe iniciar en una página principal llamada index.html que funcione como portada de acceso y contenga tres accesos directos:

1. Link al texto plano del prompt utilizado.
2. Link a la Landing Page generada por el Primer Agente.
3. Link a la Landing Page generada por el Segundo Agente.

CONTEXTO DEL PROYECTO

Este proyecto corresponde a una Práctica Formativa Obligatoria donde se utilizó un mismo prompt inicial en dos agentes de desarrollo distintos para generar dos versiones de una Landing Page.

La finalidad de esta portada es permitir comparar visualmente ambos resultados desde una única página inicial.

PFO2 - Prompt Engineering en Agentes de IA

Subtítulo:
Comparación de Landing Pages generadas con un mismo prompt inicial

Estudiante:
Monica Cáffaro

Tema de la Landing Page:
Empresa ficticia de venta de software llamada NexoSoft.

Primer Agente:
Nombre del agente: Claude
Modelo utilizado: No detectado por el entorno

Segundo Agente:
Nombre del agente: Codex
Modelo utilizado: No detectado por el entorno

ESTRUCTURA ESPERADA DEL PROYECTO

Organizar o respetar la siguiente estructura de archivos y carpetas:

/index.html
/styles.css
/prompt.txt
/agente-1/index.html
/agente-2/index.html

No modificar el código interno de las landing pages generadas por los agentes. Solo crear o modificar la portada principal del proyecto y, si es necesario, el archivo de estilos de la portada.
