PFO2 - Prompt Engineering en Agentes de IA

Comparación de Landing Pages generadas con un mismo prompt inicial

Este proyecto compara dos Landing Pages generadas por agentes de IA a partir de un único prompt inicial. El objetivo es observar diferencias de diseño, estructura y resolución autónoma entre ambos agentes.

Aclaración metodológica

Mismo prompt, dos resultados

El mismo prompt fue utilizado en ambos agentes de desarrollo. No se modifica manualmente el código generado por cada agente, con el fin de evaluar la capacidad de resolución autónoma y comparar los resultados obtenidos.
